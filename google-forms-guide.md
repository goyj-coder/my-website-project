# 구글 설문지(Google Forms) 기반 요양원 상담 신청 폼 기획안

기존 상담 채널(카카오톡 URL 등)이 없을 때 가장 확실한 무료 리드(Lead) 수집 대안인 구글 설문지 제작 가이드입니다.

## 1. 설문지 작성 전 기본 셋팅
*   **접속**: Google 계정 로그인 상태에서 [구글 설문지](https://docs.google.com/forms/)로 이동하여 '빈 양식' 생성.
*   **제목**: "행복한집 요양원 입소 상담 신청"
*   **설명**: "보호자님의 마음으로 따뜻하게 상담해 드리겠습니다. 아래 양식을 작성해 주시면 확인 후 신속하게 연락드리겠습니다. (입력하신 정보는 상담 목적으로만 안전하게 사용됩니다.)"

## 2. 권장하는 질문 문항 (필수 입력 항목 *)
사용자가 부담 없이 입력할 수 있도록 **가장 핵심적인 5개 항목**만 구성하는 것이 전환율(제출률)에 유리합니다.

1.  **[단답형] 보호자 이름 \*** 
    *   질문: "상담받으실 보호자님의 성함을 입력해 주세요."
2.  **[단답형] 연락처 \***
     *   질문: "연락받으실 휴대폰 번호를 입력해 주세요." 
     *   *(응답 확인/유효성 검사 기능으로 숫자만 입력하게 설정)*
3.  **[객관식 질문] 어르신 성별 \***
    *   옵션: 여성 전문 요양원이므로 "여성 어르신" 단일 선택지로 두거나, 혹은 "여성", "남성"으로 두어 남성 문의를 거르기. (추천: "대상 어르신은 여성이신가요? - 네 / 아니오")
4.  **[객관식 질문] 장기요양등급 보유 여부 \***
    *   질문: "어르신의 노인장기요양등급을 선택해 주세요."
    *   옵션: ① 등급 없음 (신청 예정), ② 1~2등급, ③ 3~5등급, ④ 인지지원등급
5.  **[장문형] 문의 내용 (선택 기입)**
    *   질문: "어르신의 질환(치매, 거동 불편 등)이나 궁금하신 점을 자유롭게 남겨주세요."

## 3. 홈페이지 (구글 사이트 도구) 연동 방법
1.  작성하신 설문지를 저장(퍼블리싱)합니다.
2.  `happy-house.org` 구글 사이트 도구 편집 창을 엽니다.
3.  **`입소안내 > 문의하기`** 페이지로 이동합니다.
4.  우측 패널 가장 하단에 있는 **`[Forms(설문지)]`** 메뉴를 클릭합니다.
5.  방금 만든 "행복한집 요양원 입소 상담 신청" 폼을 클릭하여 화면에 꽉 차게 삽입합니다.
