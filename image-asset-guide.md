# 홈페이지 이미지 자원 관리 내역서 (Image Asset Guide)

이 문서는 웹사이트 각 페이지에 필요한 이미지의 위치, 용도, 권장 스타일을 정리하여 나중에 실제 사진이나 아이콘을 삽입할 때 참고하기 위한 가이드입니다.

---

## 🏠 1. 홈 (Main Page)
| 위치 | 용도 | 권장 이미지 스타일 | 비고 |
| :--- | :--- | :--- | :--- |
| 메인 상단 | 히어로 배너 (캐러셀) | 요양원 전경 혹은 대표 로비 (밝고 깨끗함) | 3~5장 필요 |
| 중앙 섹션 | 메뉴 연결 버튼용 배경 | 각 메뉴(식단, 활동 등)를 상징하는 상징적 사진 | 4~6장 |
| 하단 | 상담 안내 배경 | 원장님 상담 모습 혹은 따뜻한 손잡음 | |

## 📌 촬영 및 준비 팁
- **수평/수직 맞추기**: 공간이 넓어 보이게 수평을 잘 맞춰 촬영해주세요.
- **여성 중심**: 여성 어르신 전용임을 알 수 있는 소품(꽃, 파스텔톤 가구 등) 활용 권장.
- **초상권 보호**: 어르신들의 얼굴이 직접 노출되는 것이 부담스러울 경우 뒷모습이나 손, 활동 결과물 위주로 촬영하세요.
