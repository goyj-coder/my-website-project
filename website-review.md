# WebMaster 젠 가이드라인 수정 기록 (2026.02.22)

사용자 피드백을 반영하여 `website-review.md`의 현황 파악 오류를 수정하고, 다음 단계 액션 플랜을 갱신합니다.

## 수정 사항
1.  **'왜 여성전문인가' 메뉴 위치 확인**: 기존 리뷰에서 하위 메뉴로 잘못 파악했으나, 실제 사이트(`happy-house.org`)에서는 이미 1 Depth(메인 메뉴)에 노출되어 있음을 확인 및 정정합니다.
2.  **작업 주체 명확화 (가이드라인 재확인)**: WebMaster 젠은 '구글 사이트 도구' 화면을 직접 클릭하거나 조작할 권한(연동된 API)이 없음을 명확히 인지합니다. 따라서 '직접 작업'은 불가능하며, 사용자가 가장 쉽고 빠르게 따라 할 수 있도록 **가장 정확한 메뉴 경로와 설정 방법, 코드 스니펫(복붙용)**을 제공하는 **'네비게이터(Navigator)'** 역할에 집중해야 합니다.

## 🚀 사용자 안내용: 다음 단계 실무 가이드 (Step 3: Implementation)

### 1단계: 메인 페이지(Home) 시각적 요소 보강
현재 텍스트 위주로 나열된 링크들을 보호자들이 누르기 쉬운 직관적인 요소로 변경합니다.

*   **미션 A: 텍스트 링크를 눈에 띄는 [버튼]으로 바꾸기**
    1.  구글 사이트 화면 오른쪽 패널에서 **`[삽입]`** 탭을 클릭합니다.
    2.  스크롤을 밑으로 내려 **`[버튼]`** 아이콘을 클릭하세요.
    3.  `이름` 칸에 "입소 안내 자세히 보기" 등을 적고, `링크` 칸에 연결할 페이지(입소안내)를 선택한 뒤 **[삽입]**을 누릅니다. 만들어진 버튼은 드래그해서 원하는 위치로 이동할 수 있습니다.
*   **미션 B: 요양원 사진을 멋지게 보여줄 [이미지 캐러셀] 넣기**
    1.  마찬가지로 우측 패널 **`[삽입]`** 탭에서 **`[이미지 캐러셀]`**을 찾아 클릭합니다.
    2.  `[+]` 버튼을 눌러 요양원의 깨끗한 침실, 거실, 프로그램실 등의 사진을 최소 3장 이상 업로드하고 **[삽입]**을 누르세요. 사진이 스스로 부드럽게 넘어가는 갤러리가 만들어집니다.
