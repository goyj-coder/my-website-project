## 2. 웹 기획자 젠의 집중 개선 가이드 (Action Items)

현재 구조에 아쉬운 부분들에 대해 **구글 사이트 도구 내비게이션(GNB) 구조 최적화** 및 **시각적 강조** 관점에서 피드백을 드립니다.

### 🎯 1. GNB (상단 메뉴) 구조 개편
말씀해주신 기존 구조와 현재 반영된 구성을 확인해보면 카테고리 분류가 다소 분산되어 있습니다. `project-planning.md`에서 제안한 **최적화된 7개 메뉴**로 재배치하시는 것을 권장합니다.
*   **Action 1**: `기관소개` 하위에 있던 **'왜 여성전문인가'**를 메인 메뉴(1 Depth)로 꺼내어 가장 눈에 띄게 배치하세요. (이 요양원의 가장 강력한 무기입니다!)
*   **Action 2**: `생활안내`에 묶여있던 **'공지사항'**은 성격이 다르므로 **'보호자 공간'** 하위로 이동하거나 합쳐주세요.
*   **Action 3**: '입소안내' 하위에 있는 **'자주하는 질문'**과 **'문의 하기'**를 분리하여, 사용자가 언제든 문의를 쉽게 남길 수 있도록 '상담 및 문의' 페이지로 빼는 것도 좋습니다.

### 🖼️ 2. 메인 페이지(Home) 시각적 요소 보강
현재 텍스트 링크("[입소 절차]", "[건강 식단]" 등)로 메뉴를 연결해 둔 상태로 보입니다. 보호자(4060)는 텍스트보다 직관적인 버튼이나 이미지를 선호합니다.
*   **Action**: 구글 사이트 도구 우측 패널의 **`[삽입] > [버튼]`** 기능을 활용하여 텍스트 링크를 눈에 띄는 버튼으로 바꾸세요.
*   **Action**: 메인 상단에 **`[삽입] > [이미지 캐러셀]`** 기능을 이용해 밝고 깨끗한 요양원 시설 사진 3~5장을 부드럽게 넘어가도록 배치하세요. 신뢰도가 급상승합니다!

### 💬 3. 상담 채널 연동 (가장 중요)
이메일과 전화번호 안내는 잘 되어 있지만, 최근 보호자들은 모바일에서 바로 카톡 상담을 선호합니다.
*   **Action**: 플로팅(항상 화면에 떠 있는) 형태나 메인 페이지 중앙에 **'카카오톡 1:1 상담하기' 버튼**이 추가되어야 합니다. (이 부분은 다음 단계에서 제가 직접 HTML 스니펫 코드를 짜서 드리겠습니다.)
